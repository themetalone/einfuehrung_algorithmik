Grundidee

Schau dir diese bereits sortierte Liste an.

  1. An welche Stelle muss ein Objekt mit dem Eigenschaftwert 5 eingefügt werden, damit die Liste danach immer noch sortiert ist?

  2. Wenn nun die Liste leer wäre, wie ordnet man dann das Objekt ein?

Eine Ausführung

Das Sortierverfahren

Wir wollen obige Operation nun in ein Sortierverfahren erweitern.

Schreibe eine Methode in Pseudocode, die

  1. eine Liste und einen Eigenschaftwert erhält und die Position zurückgibt, an der ein Objekt mit diesem Eigenschaftswert in der Liste stehen müsste, damit die Liste weiterhin sortiert ist.

  2. eine Liste, ein Objekt und eine Position erhält und das Objekt an der Position in die Liste einfügt

  3. eine Liste erhält und eine sortierte Liste zurück gibt — Verwende die zuvor erstellten Methoden — Wie startet das Verfahren?

Sortieren ohne Hilfsliste

Im obigen Verfahren wurden Hilfslisten verwendet. Wir wollen nun unser Sortierverfahren so abändern, dass wir ausschließlich auf der zu sortierenden Liste arbeiten.

  1. Gibt es Gründe, weswegen es vorteilhaft sein kann nur auf der Eingabeliste zu arbeiten?

  2. Schreibe das Sortierverfahren so um, dass es nur noch auf der Eingangsliste arbeitet.
